==============
BCS Gap Energy
==============

In this example, the BCS gap energy plot from the original BCS paper is
reproduced.

.. image:: /static/images/BCSGapEnergy.svg
    :align: center

The code can be found in ``./examples/bcs_gap_energy.py`` and can be run with

.. code-block:: bash

    poetry run ./examples/bcs_gap_energy.py
